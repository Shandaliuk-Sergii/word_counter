INTRODUCTION
------------

Word counter provides functionality word count of article body field and shows it on node view.
To add word counter to another content type you need just to add field_word_count to a content type,
also, the field body should exist for this content type.


INSTALLATION
------------

 * Install as you would normally install a custom Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

Use word counter have to be true http://test.docker.localhost:8272/admin/config/counter/settings
